# Contact Book (Python)

A simple command-line contact book built using Python.
This project demonstrates basic CRUD operations using dictionaries and functions.

## Features
- Add a contact
- View a contact
- Delete a contact
- Show all contacts

## Tech Stack
- Python 3

## How to Run
1. Clone the repository
2. Run the file using:
   python contact_book.py

## Learning Outcome
- Dictionary operations
- Function-based program structure
- Basic Git & GitHub workflow
